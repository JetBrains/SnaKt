Linearizer

data class Linearizer(val state: SharedLinearizationState, val seqnBuilder: SeqnBuilder, val source: KtSourceElement?, val stmtModifierTracker: StmtModifierTracker? = null) : LinearizationContext

Standard linearization context: converts an ExpEmbedding tree into a flat Viper statement sequence (Stmt.Seqn).

Each function body is linearized by creating a Linearizer rooted at a fresh SeqnBuilder. As the embedding tree is traversed, statements are accumulated via addStatement and addDeclaration; nested blocks (e.g., if-branches, while-bodies) use asBlock to produce child Stmt.Seqn nodes.

After linearization, the resulting Stmt.Seqn is passed to SsaConverter for SSA renaming.
