PropertyAccessorFunctionSignature
Link copied to clipboard
abstract class PropertyAccessorFunctionSignature(val name: SymbolicName, propertySymbol: FirPropertySymbol) : FullNamedFunctionSignature, GenericFunctionSignatureMixin

We generate very reduced methods for getters and setters. They don't have bodies or any invariants. Since after the call to getter, invariants for the result will be inhaled based on the difference between the returned TypeEmbedding and expected, we return the broadest possible type here. Types of the arguments don't matter at all, but intuitively they must be NullableAnyTypeEmbedding as well.
