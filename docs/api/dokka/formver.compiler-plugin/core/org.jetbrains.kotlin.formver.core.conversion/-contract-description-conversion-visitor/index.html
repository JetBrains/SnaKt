ContractDescriptionConversionVisitor

class ContractDescriptionConversionVisitor(ctx: ProgramConversionContext, signature: NamedFunctionSignature) : KtContractDescriptionVisitor<ExpEmbedding, ContractVisitorContext, ConeKotlinType, ConeDiagnostic>

Converts Kotlin contract { } effect declarations into ExpEmbeddings that can be used as Viper postconditions.

Implements KtContractDescriptionVisitor — the Kotlin compiler's visitor interface for the contract description AST (KtEffectDeclaration and its subtypes). Each visitX method maps one contract node type to the corresponding Viper expression.

KtEffectDeclarations are nodes generated by the compiler, and as such have no source information beyond the source of the FirEffectDeclaration. As such, we only assign position information at the top level (in getPostconditions).

Parameters

ctx

The program-level conversion context, used to embed types referenced in contracts.

signature

The signature of the function being converted, used to resolve parameter references (formal params, dispatch/extension receivers) inside the contract.
