InlineParameterResolver

class InlineParameterResolver(substitutions: Map<SubstitutedArgument, ExpEmbedding>, val labelName: String?, val defaultResolvedReturnTarget: ReturnTarget) : ParameterResolver

ParameterResolver used when inlining a function call at a call site.

Instead of looking up parameters from a FunctionSignature, it uses a caller-supplied substitutions map from SubstitutedArgument keys to the actual argument ExpEmbeddings. This ensures that each inlined call site uses fresh, uniquely-named anonymous variables rather than the callee's original parameter names, avoiding name collisions when the same function is inlined multiple times.
