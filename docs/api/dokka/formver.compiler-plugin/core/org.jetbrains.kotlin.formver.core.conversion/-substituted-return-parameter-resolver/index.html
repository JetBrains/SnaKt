SubstitutedReturnParameterResolver

class SubstitutedReturnParameterResolver(rootResolver: RootParameterResolver, substitutionContext: ReturnVarSubstitutionContext) : ParameterResolver

A ParameterResolver that wraps a RootParameterResolver and intercepts lookups for one specific symbol — the return variable used in postconditions.

All other resolution (dispatch receiver, extension receiver, other params, label, return target) is delegated to rootResolver unchanged via the by delegation.

Parameters

The underlying resolver for the function being specified.
