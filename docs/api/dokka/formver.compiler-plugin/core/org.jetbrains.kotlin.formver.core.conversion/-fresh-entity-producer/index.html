FreshEntityProducer

class FreshEntityProducer<R, S>(build: (Int, S) -> R)

A stateful factory that produces fresh instances of type R by supplying a monotonically increasing integer index together with a caller-provided seed value of type S.

The index guarantees uniqueness across all entities produced by the same instance, which is used to generate distinct Viper names for anonymous variables, return targets, scope indices, etc.
