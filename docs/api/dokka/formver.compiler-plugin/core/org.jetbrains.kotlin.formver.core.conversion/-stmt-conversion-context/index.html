StmtConversionContext

interface StmtConversionContext : MethodConversionContext

Interface for statement conversion.

Extends MethodConversionContext with the mutable, statement-level state needed while walking a FIR function body: the current when-subject variable, safe-call subject caching, active exception catch labels, and loop/scope management.

All conversion of FIR statements and expressions flows through convert, which dispatches to StmtConversionVisitor.

Naming convention:

  • Functions that return a new StmtConversionContext should describe what change they make (addResult, removeResult...)

  • Functions that take a lambda to execute should describe what extra state the lambda will have (withResult...)
