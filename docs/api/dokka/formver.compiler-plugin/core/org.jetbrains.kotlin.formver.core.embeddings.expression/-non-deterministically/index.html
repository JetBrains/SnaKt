NonDeterministically

data class NonDeterministically(val exp: ExpEmbedding) : UnitResultExpEmbedding, DefaultDebugTreeViewImplementation

Wraps exp in a non-deterministic branch: lowered to a Viper if (*) so the expression may or may not execute. Used to model optional side effects without committing to a path.
