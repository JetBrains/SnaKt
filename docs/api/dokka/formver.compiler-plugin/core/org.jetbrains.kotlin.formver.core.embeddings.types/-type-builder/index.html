TypeBuilder

class TypeBuilder

Builder for a TypeEmbedding.

We split most of the work into PretypeBuilder, which builds the type modulo nullability (and potentially other flags). As a result, the builder does not contain the full building state at any point, though a TypeBuilder, PretypeBuilder pair does.
