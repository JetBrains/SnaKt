BuiltInMethod

abstract class BuiltInMethod(name: SymbolicName, pos: Position = Position.NoPosition, info: Info = Info.NoInfo, trafos: Trafos = Trafos.NoTrafos) : Method

Base class for internally generated helper methods (e.g. allocation stubs, runtime-type helpers) that are part of the Viper encoding infrastructure rather than user code.

Built-in methods are excluded from the abbreviated Viper dump so that log output stays focused on user-visible methods.
