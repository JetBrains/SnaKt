FormalVerificationPluginNames

object FormalVerificationPluginNames

String constants shared between the CLI processor, the Gradle plugin, and the compiler plugin to identify the plugin and its configuration options by name.

Using this object ensures that option names in the CLI argument parser, the Gradle DSL, and the plugin registration all stay in sync. Any rename of a configuration option must be reflected here, in the CLI parser, and in the Gradle extension simultaneously.
